Pour publier mon projet réalisé avec Bootstrap, j’ai utilisé le site GitHub et la méthode simple du drag and drop (glisser-déposer).
D’abord, je me suis connecté à mon compte GitHub et j’ai créé un nouveau dépôt en cliquant sur "New". J’ai donné un nom à mon projet et je l’ai mis en public.
Ensuite, j’ai cliqué sur "Upload files" pour ajouter mes fichiers. J’ai ouvert le dossier de mon projet sur mon ordinateur, puis j’ai sélectionné tous les fichiers (HTML, CSS, images) et je les ai glissés directement sur GitHub.
Après le téléchargement, j’ai écrit un message de validation et cliqué sur "Commit changes" pour enregistrer.
Enfin, mon projet était en ligne et accessible grâce au lien GitHub.
